
The Expense Tracker System is a web app that helps people record and manage their expenses in a clear and easy way. Old methods like notebooks or spreadsheets can take a lot of time and often lead to mistakes. This system makes the process faster and more accurate by automating it.

The app is built with (PHP, MySQL, HTML, CSS, and JavaScript) and runs on an Apache server. It allows users to log in securely, add, edit, or delete expenses, and see reports of their spending by day, month, or year.

The main goals of the system are:

Replace manual tracking with a digital tool

Provide quick and accurate reports

Keep data safe and secure

Make the app simple to use on any modern browser

Application Functionalities

User Accounts: Register, login, logout, password reset.

Expense Management: Add, edit, delete, and view expenses.

Reports: Daily, monthly, yearly summaries with category breakdowns.

Profile: Update details and change password.

System: Secure, responsive, and fast performance.
