# Docs Site

This folder contains the GitHub Pages version of the project report using the `Just the Docs` theme.

## Publish Steps

1. Push your project to GitHub.
2. Open `Settings > Pages` in the repository.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/docs`.
5. Save and wait for GitHub Pages to build the site.

Your report will then be available at:

`https://zagrosbaban1.github.io/Expense-Tracker-System/`
